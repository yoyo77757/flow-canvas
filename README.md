Create visual flow diagrams by clicking anywhere on canvas. Auto-detects keywords and styles nodes. Drag to connect nodes and build flowcharts.

## Features

- Click to add nodes; drag to connect with lines
- Keywords (e.g. API, database, user) get automatic styling
- No formal syntax; ideal for quick sketches
- Export or share diagram
- Great for architecture or process flows

## Why use this tool

Sketch a flow or architecture without learning diagram syntax. Faster than opening a heavy app; good for standups or docs.
